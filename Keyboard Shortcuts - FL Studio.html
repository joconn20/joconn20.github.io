Keyboard Shortcuts

by John O'Connor III

FL Studio

Shortcut Function
Ctrl + C Copy the selected item to the clipboard.
Ctrl + X Cut the selected item and move it to the clipboard.
Ctrl + V Paste the content from the clipboard into the current location.
Ctrl + Z Undo the last action.
Ctrl + Y Redo the last undone action.
Ctrl + A Select all items in the current document or window.
Ctrl + S Save the current document or file.
Ctrl + P Open the print dialog to print the current document.
Ctrl + F Open the search or find feature to locate specific text.
Alt + Tab Switch between open applications.
Alt + F4 Close the active window or application.
Windows Key + D Show or hide the desktop.
Windows Key + L Lock the computer.
Ctrl + Shift + Esc Open the Task Manager.
Ctrl + Alt + Delete Open the security options screen (e.g., lock, switch user, Task Manager).
F1 Open the help menu for the active application.
F2 Rename the selected item.
F5 Refresh the active window or webpage.
Ctrl + N Open a new window or document.
Ctrl + O Open an existing file or document.
Ctrl + B Apply bold formatting to the selected text.
Ctrl + I Apply italic formatting to the selected text.
Ctrl + U Apply underline formatting to the selected text.
Ctrl + Left Arrow Move the cursor to the beginning of the previous word.
Ctrl + Right Arrow Move the cursor to the beginning of the next word.
Ctrl + Home Move the cursor to the beginning of the document.
Ctrl + End Move the cursor to the end of the document.
Ctrl + Backspace Delete the word to the left of the cursor.
Ctrl + Delete Delete the word to the right of the cursor.
Ctrl + Shift + N Create a new folder in File Explorer.
Windows Key + E Open File Explorer.
Windows Key + R Open the Run dialog box.
Windows Key + I Open the Settings window.
Windows Key + S Open the search bar.
Windows Key + Tab Open Task View to see all open windows and desktops.
Windows Key + Up Arrow Maximize the current window.
Windows Key + Down Arrow Minimize the current window.
Windows Key + Left Arrow Snap the current window to the left half of the screen.
Windows Key + Right Arrow Snap the current window to the right half of the screen.
